# DevSecOps → SIEM + SOAR Integration

This project demonstrates:

- Flask application
- Hardened Docker image
- Trivy SBOM & SARIF scanning
- GitHub Code Scanning upload
- Send SBOM + SARIF to SIEM
- SIEM stack: OpenSearch + Dashboards + Filebeat
- SOAR stack: TheHive + Cortex

This is a full DevSecOps → Detection → Response pipeline.
